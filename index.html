Team Radar

Shows the current mood of each team member on one radiator web page. Especially with multi-site teams this aims to bring the whole team mentally a little bit closer despite the geographical distance (which always sucks) between sites.

Contains an IRC bot for posting mood updates, WebSockets for pushing updates to all browser clients in real-time, Node.js, Express.js, Socket.io, Scss... So it's cool and you should use it!!1

Installation instructions and source code in github.
